# Shooting Gale

A Simple Board Game developed using React.

## Setting up

In the project directory,

(1) Run `npm install`.

(2) Run `npm start`.

## Run the app in the development mode

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
